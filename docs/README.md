# 🍳 What's Your Community Management Flavour?

We are developing an interactive quiz that helps people identify what their dominant skills are as part of a community.
Specifically, we hope they will identify with the curated personas of Research Community Managers (RCMs).

We created 8 scenario-based questions to reveal which of the five RCM skill areas feels most natural to you, presented through a fun theme - what kind of cook are you.

### The 5 Personas

| Persona | Skill Area | Emoji |
|---------|-----------|-------|
| Influencer Cook | Communications | 📢 |
| Community Cook | Engagement | 🤝 |
| Head Chef | Strategic Leadership | 🧠 |
| Hostess with the Mostest | Programme Management | 📋 |
| Nerd Cook | Technical Skills | 💻 |

## Input files

| File | Description |
|------|-------------|
| `persona-questions.csv` | Source questions with answer options mapped to each persona |

| `persona-descriptions.csv` | Persona description for display |
| `RCMSkills.csv` | RCM skill matrix used to ground the quiz options |
| `SKILL.md` | Copilot CLI skill — use this to regenerate all outputs from new source files |

## Disclaimer: Use of Claude

### How to regenerate outputs from new source files

This project includes a `SKILL.md` — a [GitHub Copilot CLI](https://docs.github.com/copilot/concepts/agents/about-copilot-cli) skill that automates everything.

### Prerequisites

- [GitHub Copilot CLI](https://docs.github.com/copilot/concepts/agents/about-copilot-cli) installed

### Steps

1. Copy `SKILL.md` into `~/.copilot/skills/rcm-quiz-generator/SKILL.md`
2. Navigate to the folder containing your source files
3. Launch Copilot CLI: `copilot`
4. Run `/skills` and enable `rcm-quiz-generator`
5. Ask Copilot to run the skill

Copilot will generate:
- `quiz.html` — interactive quiz with results, strengths, weaknesses, and recommendations
- `quiz-options.csv` — all questions and options in wide format
- `persona-strengths-weaknesses.csv` — one strength, weakness, and recommendation per persona

## Output files

| File | Description |
|------|-------------|
| `persona-questions.csv` | 5 answer options generated by combining personas per question using the RCM Skills Frameowkr from Sharan et. al. (2024) in spreadsheet format |
| `persona-strengths-weaknesses.csv` | One strength, one weakness, one recommendation, and a best collaborator per persona |

These files were manually reviewed and improved by the team.
No AI was used for community manager persona.


## Source material

Quiz options and persona descriptions are grounded in the Research Community Manager skills framework from the following publication (by the RCM Coop team), covering the full project lifecycle from initiation through to sustainability.
